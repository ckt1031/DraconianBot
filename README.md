### Initiation

- Environmental Variables (.env)

```
TOKEN=XXXXX.ABCDEFG
CLIENT_ID=XXXXXXXXXXXXXXXXXX
MONGODB_URL=XXXXXXXXXXXXXXXX
```

- Install Dependencies

```
npm install
```

- Build & Compile to JS (With SWC compiler)

```
npm run build
```

- Copy Config (For FIRST / AFTER SYNCED)
  > Save the config first!

```
npm sync
```

- Start

```
npm run start
```
